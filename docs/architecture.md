# AI Life Replay Architecture

## Flow
Upload photo -> backend -> Cloudinary -> AI summary -> MongoDB -> timeline -> diary -> search.

## Hackathon focus
- Make the upload flow feel instant.
- Make timeline and search feel smart.
- Make the memory graph the unique differentiator.

## Team split
- Frontend: React screens and polished demo.
- Backend: FastAPI endpoints and MongoDB data shape.
- AI: Groq prompts for summary, diary, search, insights.
- Integration: Cloudinary, auth, deployment, demo script.
