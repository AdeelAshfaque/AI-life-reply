# Groq Prompts

## Photo summary
You are an assistant that turns a photo into a memory. Return:
- title
- concise description
- tags
- mood
- people
- location

## Diary generation
Summarize the user's day from ordered memories in one friendly paragraph.

## Search answer
Answer the user's question using only stored memories. If possible, mention date, place, and time.

## Weekly insights
Aggregate counts by activity type, people, and location. Keep it concise and readable.
